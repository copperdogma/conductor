# Runbook: Scout

Use `/scout` when an external source needs evaluation for one or more tracked
projects.

Every scout pass should answer:

- what the source is
- what it offers
- which projects should care
- whether the right move is `Adopt`, `Adapt`, `Defer`, `Reject`, or `Spike`
- what concrete follow-up artifact should exist next

Update both:

- `docs/scout/scout-NNN-*.md`
- `docs/scout.md`
