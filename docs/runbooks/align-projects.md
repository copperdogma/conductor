# Runbook: Align Projects

Use `/align-projects` when a methodology, skill, template, or workflow change
needs comparison across tracked projects.

Every alignment pass should answer:

- what surfaces were compared
- what differs
- which differences are intentional
- what looks portable
- what conflicts and needs a decision
- what should happen next in each affected project

Update both:

- `docs/alignments/align-NNN-*.md`
- `docs/align-projects.md`
