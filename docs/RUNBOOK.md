# Conductor Runbook

## Core commands

```bash
make methodology-compile
make methodology-check
make skills-sync
make skills-check
make lint
make test
```

## Typical loop

1. Add raw notes or links to `inbox.md`
2. Run `/triage`
3. Convert worthy work into a story or scout/alignment log
4. Build and validate the resulting artifact
5. Close the story once the supervisor output is real and inspectable

## Specialized loops

- `/align-projects` — compare tracked-project infrastructure surfaces
- `/scout` — assess external sources for project relevance
- `/setup-methodology` — refresh this project's own methodology package
